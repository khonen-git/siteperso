## Résumé

J'ai réalisé une base de données avec MySQL. Le code SQL de la base de données a été réalisé via le logiciel SQL Power Architect après avoir fait un dictionnaire des données sur Excel. J'ai ensuite fait des requêtes SQL pour obtenir des informations sur les données contenues dans la base.

## Contexte

Le projet est une introduction aux bases de données et à SQL. Nos jeux des données sont des contrats clients d'une assurance habitation et des informations du référentiel géographiques extraites du site data.gouv.fr. 

## Objectifs

- Réaliser un dictionnaire des données sur Excel avec les noms et types de variables, les contraintes, les desciptions, etc.
- Faire un schéma relationnel via le logiciel SQL Power Architect.
- Créer la base de données dans MySQL à partir du code généré par le logiciel SQL Power Architect
- Requêtez la base de données avec SQL
- Faire une synthèse et présentez les résultats

## Technologies 

- SQL Power Architect
- MySQL
- Excel

## Langages 

- SQL

## Résultats

J'ai une base de données fonctionnelle en local dans MySQL que je peux requêtez facilement et j'ai établi un rapport sur l'ensemble du projet.

## Compétences acquises

- Maîtrise du logiciel SQL Power Architect
