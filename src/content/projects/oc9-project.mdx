# OpenClassrooms Data Analyst - Projet 9 : Analysez les ventes d'une librairie avec Python

## Résumé

Ce projet est une analyse avancée des ventes d'une librairie. L'entreprise, qui est à l'origine physique, a décidé d'étendre son marché avec un site de vente en ligne, c'est les ventes du site en ligne qui seront analysés. Ces analyses serviront plus tard de comparaison avec la librairie physique. 

## Objectifs

- Analyser les indicateurs de ventes
- Calculer des moyennes mobiles pour lisser les variations des indicateurs
- Analyser les références et les Comparer
- Analyser les clients B2B
- Formuler des hypothèses statistiques et réaliser des tests
- Analyser des corrélations

## Langages
