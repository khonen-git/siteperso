## Résumé

J'ai réalisé une base de donnnées immobilière conforme au RGPD qui permettra par la suite de créer une modèle pour mieux prévoir le prix des ventes des biens immobiliers. Nous avons des données extraites du site open data des Deamandes de valeurs foncières (DVF), des données de l'INSEE avec les résultats de recensements de la population et des données de data.gouv sur les référentiels géographiques. J'ai donc réalisé un dictionnaire des données pour documenter les données importantes puis réalisé un schéma en faisant attention à ce que celui-ci suive la norme 3NF. J'ai aussi pris soin à ce que les données soit conforme aux RGPD et respecte la vie privée des individus liés à ces données. Ensuite, j'ai testé cette base de données via des requêtes et présenté mon travail.

## Objectifs

- Réaliser une dictionnaire des données sur Excel
- Créer un schéma normalisé 3NF via SQl Power Architect
- Vérifier et justifier la conformité RGPD
- Créer la base de données dans MySQL
- Importer les données dans la base de données et faire les requêtes SQL pour valider la base de données
- Présenter les résultats et l'utilisation de la base de données

## Résultats

Nous avons une base de données immobilière conforme RGPD qui peut être utilisée pour faire des prévisions sur l'évolution du marché de l'immobilier.
