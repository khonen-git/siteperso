# OpenClassrooms Data Analyst - Projet 3 : Requêtez une base de données avec SQL

## Résumé

Ce projet est la création d'un tableau de bord avec Power BI permettant de visualiser de gérer l'avancement des projets d'une entreprise internationale du soin bucco-dentaire. Avant de commencer la réalisation du dahsboard, j'ai analysé les besoins utilisateurs dans la note de cadrage et formulé des users stories pour chaque utilisateur que j'ai complété dans le Product Strategy Canvas. J'ai ensuite fait un mockup pour esquisser la forme des vues du dashboard et avoir une première idée de sa réalisation. J'ai ensuite analysé les données dans Power Query Editor, appliqué certains traitements et construit un schéma en étoile pour optimiser les performances. Ce n'est qu'apprès ces étapes que j'ai commencé à réalisé les vues du dashboard : mise en place de graphiques à partir de KPIs qui répondent aux users stories, création d'indicateurs calculés avec des formules DAX, création d'infobulles pour obtenir des informations détaillés, ajout de possibilités de filtres et de recherches. J'ai aussi créé des rôles avec des restrictions pour les différentes types d'utilisateurs. J'ai fini en documentant mes vues et la méthode pour mettre à jour les données du dahsboard.

## Objectifs

- Identifier les besoins utilisateurs
- Formuler des users stories
- Créer 3 vues à différente échelle pour visualiser l'avancement des projets
- Créer des infobulles pour avoir des informations détaillées
- Créer des rôles avec des droits d'accès pour les utilisateurs
- Documenter les vues et les graphiques
- Documenter la méthodologie pour mettre à jour les données
