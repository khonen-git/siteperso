# Équations aux dérivées partielles

<MdxCard title="Définition">
Une **équation aux dérivées partielles** (EDP) relie une fonction inconnue <MathInline>{"u(x_1, \\ldots, x_n)"}</MathInline> à ses **dérivées partielles** :

<MathBlock>{"F\\left(x, u, \\frac{\\partial u}{\\partial x_i}, \\frac{\\partial^2 u}{\\partial x_i \\partial x_j}, \\ldots\\right) = 0"}</MathBlock>

Contrairement aux [EDO](/knowledge/mathematics/analysis/ordinary-differential-equations), plusieurs variables **indépendantes** coexistent — typiquement le **temps** <MathInline>{"t"}</MathInline> et des variables **d'espace** ou de **marché** (prix du sous-jacent, taux, etc.).

Une solution est en général déterminée par des **conditions aux limites** (sur le bord du domaine) et/ou des **conditions initiales** (en <MathInline>{"t = 0"}</MathInline>).
</MdxCard>

## Intuition

<MdxCard>
Une EDP décrit un phénomène qui varie **simultanément** dans le temps et dans l'espace (ou entre plusieurs facteurs). La solution <MathInline>{"u"}</MathInline> n'est plus une courbe mais une **surface** ou un champ.

- **Elliptique** : état d'équilibre, pas de direction privilégiée de propagation (Laplace).
- **Parabolique** : diffusion / dissipation dans le temps (chaleur, Black-Scholes).
- **Hyperbolique** : propagation d'ondes à vitesse finie (équation des ondes).
</MdxCard>

## Méthodes de résolution

<MdxCard title="Analytiques">
- **Séparation des variables** (Fourier, Sturm-Liouville).
- **Transformée de Fourier / Laplace** pour équations linéaires sur domaines simples.
- **Fonctions fondamentales (Green)** : solution pour une source ponctuelle.
- **Black-Scholes** : transformation en équation de la chaleur via changement de variables.
</MdxCard>

<MdxCard title="Numériques">
- **Différences finies** : discrétisation de <MathInline>{"u_t"}</MathInline>, <MathInline>{"u_{xx}"}</MathInline> sur une grille — voir [Calcul numérique](/knowledge/mathematics/numerical-methods).
- **Éléments finis** : base variée sur domaines complexes.
- **Monte Carlo** : alternative probabiliste via Feynman-Kac (espérance sous mesure risque-neutre).

Le choix dépend du domaine, de la dimension et des conditions aux limites.
</MdxCard>

## Lien probabilités : Feynman-Kac

<MdxCard>
Pour certaines EDP paraboliques, la solution s'écrit comme une **espérance** :

<MathBlock>{"u(x, t) = \\mathbb{E}\\left[ e^{-\\int_t^T r(s)\\,ds} \\, g(X_T) \\, \\middle| \\, X_t = x \\right]"}</MathBlock>

où <MathInline>{"X"}</MathInline> suit une EDS liée aux coefficients de l'EDP. C'est le pont entre :

- **pricing PDE** (Black-Scholes backward) ;
- **pricing Monte Carlo** (simulation risque-neutre).

Les deux approches doivent donner le même prix sous les mêmes hypothèses — utile pour validation croisée.
</MdxCard>

## Applications

<MdxCard>
- **Finance quantitative** : pricing d'options ([Black-Scholes](/knowledge/quantitative-finance/options-derivatives/black-scholes)), modèles de vol locale, calibration de surfaces.
- **Physique** : diffusion, mécanique des fluides, électromagnétisme.
- **Ingénierie** : transfert thermique, optimisation de forme.
- **Image / ML** : diffusion comme modèle de bruit (score-based models) — même formalisme parabolique.

Les [EDO](/knowledge/mathematics/analysis/ordinary-differential-equations) sont le cas à une seule variable indépendante ; les EDP ajoutent la dimension spatiale ou multi-facteurs.
</MdxCard>
