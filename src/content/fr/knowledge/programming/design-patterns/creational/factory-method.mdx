# Factory Method

<MdxCard>
Définir une interface de création et laisser les sous-classes choisir l'objet concret.
</MdxCard>

## Intention

Découpler le code client des classes concrètes instanciées.

## Exemple Python

<CodeBlock language="python">
{`from abc import ABC, abstractmethod

class Dialog(ABC):
    @abstractmethod
    def create_button(self):
        pass

    def render(self):
        btn = self.create_button()
        print(f"Rendering {btn}")

class WindowsDialog(Dialog):
    def create_button(self):
        return "WindowsButton"

class WebDialog(Dialog):
    def create_button(self):
        return "HTMLButton"

for dialog in [WindowsDialog(), WebDialog()]:
    dialog.render()`}
</CodeBlock>

<ConsoleOutput title="Sortie console">
Rendering WindowsButton
Rendering HTMLButton
</ConsoleOutput>

## Quand l'utiliser

La bibliothèque ne connaît pas à l'avance les types concrets à instancier.
