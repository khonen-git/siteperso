## Exemple Python

<CodeBlock language="python">
{`class DataMiner:
    def mine(self, path):
        data = self.open(path)
        raw = self.extract(data)
        result = self.parse(raw)
        self.close(data)
        return result

    def open(self, path):
        print(f"Open {path}")
        return path

    def extract(self, data):
        raise NotImplementedError

    def parse(self, raw):
        print(f"Parse {raw}")
        return raw

    def close(self, data):
        print(f"Close {data}")

class PDFMiner(DataMiner):
    def extract(self, data):
        print("Extract PDF bytes")
        return "pdf-bytes"

print(PDFMiner().mine("report.pdf"))`}
