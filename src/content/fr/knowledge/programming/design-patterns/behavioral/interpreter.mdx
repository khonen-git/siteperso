# Interpreter

<MdxCard>
Définir une grammaire et un interprète pour des expressions d'un langage simple.
</MdxCard>

## Intention

Évaluer des règles ou formules configurables (filtres, permissions).

## Exemple Python

<CodeBlock language="python">
{`class Context:
    def __init__(self, variables):
        self.variables = variables

class Expression:
    def interpret(self, context):
        raise NotImplementedError

class Number(Expression):
    def __init__(self, value):
        self.value = value

    def interpret(self, context):
        return self.value

class Variable(Expression):
    def __init__(self, name):
        self.name = name

    def interpret(self, context):
        return context.variables[self.name]

class Add(Expression):
    def __init__(self, left, right):
        self.left, self.right = left, right

    def interpret(self, context):
        return self.left.interpret(context) + self.right.interpret(context)

expr = Add(Variable("x"), Number(10))
ctx = Context({"x": 5})
print(expr.interpret(ctx))`}
</CodeBlock>

<ConsoleOutput title="Sortie console">
15
</ConsoleOutput>

## Quand l'utiliser

Grammaire simple et stable, peu de règles de langage.
