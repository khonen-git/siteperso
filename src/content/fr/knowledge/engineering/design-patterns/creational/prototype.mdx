Créer de nouveaux objets en copiant un prototype existant.
</MdxCard>

## Intention

Cloner est moins coûteux que reconstruire depuis zéro (configurations lourdes).

## Exemple Python

<CodeBlock language="python">
{`import copy

class Config:
    def __init__(self, settings):
        self.settings = settings

original = Config({"theme": "dark", "lang": "fr"})
clone = copy.deepcopy(original)
clone.settings["lang"] = "en"

print(original.settings)
print(clone.settings)
print(original.settings is clone.settings)`}
</CodeBlock>

<ConsoleOutput
  title="Sortie console"
  output={`{'theme': 'dark', 'lang': 'fr'}
{'theme': 'dark', 'lang': 'en'}
False`}
/>

## Quand l'utiliser

Instanciation coûteuse ou beaucoup d'états proches à dupliquer.
