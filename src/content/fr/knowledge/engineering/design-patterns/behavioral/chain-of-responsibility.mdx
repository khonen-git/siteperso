Passer une requête le long d'une chaîne de handlers jusqu'à ce qu'un la traite.
</MdxCard>

## Intention

Découpler l'émetteur des traitements (support, middleware, validation).

## Exemple Python

<CodeBlock language="python">
{`class Handler:
    def __init__(self):
        self.next = None

    def set_next(self, handler):
        self.next = handler
        return handler

    def handle(self, amount):
        if self.next:
            return self.next.handle(amount)
        return "Rejected"

class Manager(Handler):
    def handle(self, amount):
        if amount <= 1000:
            return f"Manager approved {amount}"
        return super().handle(amount)

class Director(Handler):
    def handle(self, amount):
        if amount <= 5000:
            return f"Director approved {amount}"
        return super().handle(amount)

chain = Manager()
chain.set_next(Director())
print(chain.handle(500))
print(chain.handle(3000))`}
</CodeBlock>

<ConsoleOutput title="Sortie console">
Manager approved 500
Director approved 3000
</ConsoleOutput>

## Quand l'utiliser

Plusieurs objets peuvent traiter une requête et l'ordre compte.
