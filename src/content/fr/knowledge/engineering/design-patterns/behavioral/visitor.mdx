## Intention

Ajouter des opérations sur une hiérarchie stable sans la modifier.

## Exemple Python

<CodeBlock language="python">
{`class Circle:
    def __init__(self, radius):
        self.radius = radius

    def accept(self, visitor):
        return visitor.visit_circle(self)

class Square:
    def __init__(self, side):
        self.side = side

    def accept(self, visitor):
        return visitor.visit_square(self)

class AreaVisitor:
    def visit_circle(self, circle):
        return 3.14 * circle.radius ** 2

    def visit_square(self, square):
        return square.side ** 2

shapes = [Circle(2), Square(3)]
visitor = AreaVisitor()
for s in shapes:
    print(round(s.accept(visitor), 2))`}
</CodeBlock>

## Quand l'utiliser

Nouvelles opérations fréquentes sur une structure d'objets rarement modifiée.
