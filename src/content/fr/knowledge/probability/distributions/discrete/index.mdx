# Distributions discrètes

<MdxCard>
Les **lois discrètes** modélisent des variables à support fini ou dénombrable. Les trois lois fondamentales ci-dessous forment une chaîne : un essai ([Bernoulli](/knowledge/probability/distributions/discrete/bernoulli)), la somme de <MathInline>n</MathInline> essais ([binomiale](/knowledge/probability/distributions/discrete/binomial)), puis la limite de comptage rare ([Poisson](/knowledge/probability/distributions/discrete/poisson)).
</MdxCard>
