# Prototype

<MdxCard>
Create new objects by copying an existing prototype.
</MdxCard>

## Intent

Cloning can be cheaper than rebuilding from scratch.

## Python example

<CodeBlock language="python">
{`import copy

class Config:
    def __init__(self, settings):
        self.settings = settings

original = Config({"theme": "dark", "lang": "fr"})
clone = copy.deepcopy(original)
clone.settings["lang"] = "en"

print(original.settings)
print(clone.settings)
print(original.settings is clone.settings)`}
</CodeBlock>

<ConsoleOutput
  title="Console output"
  output={`{'theme': 'dark', 'lang': 'fr'}
{'theme': 'dark', 'lang': 'en'}
False`}
/>

## When to use

Expensive setup or many similar object states to duplicate.
