## Intent

Control access to a shared resource (config, connection pool, logger).

## Python example

<CodeBlock language="python">
{`class Singleton:
    _instance = None

    def __new__(cls):
        if cls._instance is None:
            cls._instance = super().__new__(cls)
        return cls._instance

a = Singleton()
b = Singleton()
print(a is b)
print(id(a), id(b))`}

## When to use

Only one instance must exist process-wide. In modern Python, prefer modules or dependency injection over classic Singleton.
