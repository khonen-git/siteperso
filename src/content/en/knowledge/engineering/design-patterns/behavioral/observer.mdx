## Intent

Decouple event sources from listeners (UI, domain events).

## Python example

<CodeBlock language="python">
{`class Stock:
    def __init__(self, symbol):
        self.symbol = symbol
        self.price = 0
        self._observers = []

    def attach(self, observer):
        self._observers.append(observer)

    def set_price(self, price):
        self.price = price
        for obs in self._observers:
            obs.update(self)

class Investor:
    def update(self, stock):
        print(f"Alert {stock.symbol} -> {stock.price}")

stock = Stock("ACME")
stock.attach(Investor())
stock.set_price(120)`}
</CodeBlock>

<ConsoleOutput title="Console output">
Alert ACME -> 120
</ConsoleOutput>

## When to use

A change must propagate to a variable number of listeners.
