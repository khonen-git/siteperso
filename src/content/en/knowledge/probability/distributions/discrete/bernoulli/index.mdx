# Bernoulli distribution

<MdxCard title="Definition">
The **Bernoulli distribution** describes the outcome of a **single trial** with two outcomes: success (coded 1) or failure (coded 0). It is the simplest indicator variable and the building block of [binomial](/knowledge/probability/distributions/discrete/binomial) and counting models.
</MdxCard>

      </TableRow>
      <TableRow>

      </TableRow>
      <TableRow>
        <TableCell>Integer moments</TableCell>
        <TableCell><MathInline>{"\\mathbb{E}[X^k] = p"}</MathInline> for <MathInline>{"k \\geq 1"}</MathInline></TableCell>
        <TableCell>Since <MathInline>{"X^k = X"}</MathInline> on <MathInline>{"\\{0,1\\}"}</MathInline></TableCell>
      </TableRow>
      <TableRow>
        <TableCell>Maximum variance</TableCell>
        <TableCell><MathInline>{"\\max_p \\mathrm{Var}(X) = \\frac{1}{4}"}</MathInline></TableCell>
        <TableCell>Reached at <MathInline>{"p = 1/2"}</MathInline>; bounds single-trial uncertainty</TableCell>
      </TableRow>
      <TableRow>
        <TableCell>Zero product</TableCell>
        <TableCell><MathInline>{"\\mathbb{E}[X(1-X)] = 0"}</MathInline></TableCell>
        <TableCell><MathInline>{"X"}</MathInline> and <MathInline>{"1-X"}</MathInline> are never simultaneously nonzero</TableCell>
      </TableRow>
      <TableRow>
        <TableCell>MGF of sums</TableCell>
        <TableCell><MathInline>{"M_{\\sum X_i}(t) = (q + p e^t)^n"}</MathInline></TableCell>
        <TableCell>For <MathInline>n</MathInline> i.i.d. variables; identifies the binomial</TableCell>
      </TableRow>
      <TableRow>
        <TableCell>Degenerate cases</TableCell>
        <TableCell><MathInline>{"p = 0 \\Rightarrow X = 0"}</MathInline>, <MathInline>{"p = 1 \\Rightarrow X = 1"}</MathInline></TableCell>

## Applications

<div className="grid gap-4 md:grid-cols-2">

<MdxCard>

### Default modeling

A loan defaulting (yes/no) over a horizon is often modeled as a Bernoulli with parameter PD (probability of default).

</MdxCard>

<MdxCard>

### Binary classification

Classifier output (positive/negative), marketing conversion, binary signal: the target or prediction error is naturally Bernoulli.

</MdxCard>

<MdxCard>

### Independent trials

Coin flips, noisy bit transmission, winning/losing trades under i.i.d. assumptions: each repetition is a Bernoulli trial.

</MdxCard>

<MdxCard>

### Building block for counting laws

Sums of i.i.d. Bernoulli variables yield the binomial; in the rare regime (large <MathInline>n</MathInline>, small <MathInline>p</MathInline>), the [Poisson law](/knowledge/probability/distributions/discrete/poisson) appears as a limit.

</MdxCard>

</div>

## Statistical inference

<MdxCard>
  <Table>
    <TableHeader>
      <TableRow>
        <TableHead>Concept</TableHead>
        <TableHead>Formula</TableHead>
        <TableHead>Description</TableHead>
      </TableRow>
    </TableHeader>
    <TableBody>
      <TableRow>
        <TableCell>Sample</TableCell>
        <TableCell><MathInline>{"X_1, \\ldots, X_n \\stackrel{\\mathrm{i.i.d.}}{\\sim} \\mathrm{Bernoulli}(p)"}</MathInline></TableCell>
        <TableCell><MathInline>n</MathInline> observed trials; inference targets proportion <MathInline>p</MathInline></TableCell>
      </TableRow>
      <TableRow>
        <TableCell>MLE estimator</TableCell>
        <TableCell><MathInline>{"\\hat{p} = \\bar{X}_n = \\frac{1}{n}\\sum_{i=1}^n X_i"}</MathInline></TableCell>
        <TableCell>Empirical success rate; unbiased, consistent, efficient</TableCell>
      </TableRow>
      <TableRow>
        <TableCell>Estimator variance</TableCell>
        <TableCell><MathInline>{"\\mathrm{Var}(\\hat{p}) = \\frac{p(1-p)}{n}"}</MathInline></TableCell>
        <TableCell>Decays as <MathInline>{"1/n"}</MathInline>; largest at <MathInline>{"p = 1/2"}</MathInline></TableCell>
      </TableRow>
      <TableRow>
        <TableCell>Wald interval</TableCell>
        <TableCell><MathInline>{"\\hat{p} \\pm z_{1-\\alpha/2} \\sqrt{\\frac{\\hat{p}(1-\\hat{p})}{n}}"}</MathInline></TableCell>
        <TableCell>Normal approximation; acceptable when <MathInline>{"np"}</MathInline> and <MathInline>{"n(1-p)"}</MathInline> are large enough</TableCell>
      </TableRow>
      <TableRow>
        <TableCell>Clopper-Pearson interval</TableCell>
        <TableCell><MathInline>{"\\left[F_{\\mathrm{Beta}}^{-1}\\left(\\frac{\\alpha}{2}; k, n-k+1\\right),\\; F_{\\mathrm{Beta}}^{-1}\\left(1-\\frac{\\alpha}{2}; k+1, n-k\\right)\\right]"}</MathInline></TableCell>
        <TableCell>Exact proportion interval with <MathInline>{"k = \\sum X_i"}</MathInline></TableCell>
      </TableRow>
      <TableRow>
        <TableCell>Proportion Z-test</TableCell>
        <TableCell><MathInline>{"Z = \\frac{\\hat{p} - p_0}{\\sqrt{p_0(1-p_0)/n}} \\approx \\mathcal{N}(0,1)"}</MathInline></TableCell>
        <TableCell>Under <MathInline>{"H_0 : p = p_0"}</MathInline> and large <MathInline>n</MathInline></TableCell>
      </TableRow>
      <TableRow>
        <TableCell>Conjugate prior (Bayesian)</TableCell>
        <TableCell><MathInline>{"p \\sim \\mathrm{Beta}(\\alpha, \\beta) \\Rightarrow p \\mid \\mathbf{x} \\sim \\mathrm{Beta}(\\alpha + k, \\beta + n - k)"}</MathInline></TableCell>
        <TableCell>The beta law is conjugate to Bernoulli</TableCell>
      </TableRow>
    </TableBody>
  </Table>
</MdxCard>

<MdxCard>
  <div className="space-y-4">
    <h3 className="text-xl font-medium">Important remarks</h3>
    <ul className="list-disc pl-6 space-y-2">
      <li>A single trial (<MathInline>{"n = 1"}</MathInline>) cannot estimate <MathInline>p</MathInline> with a margin of error; inference practically concerns an i.i.d. Bernoulli sample.</li>
      <li>The Wald interval can yield bounds outside <MathInline>{"[0,1]"}</MathInline> for <MathInline>p</MathInline> near 0 or 1 and moderate <MathInline>n</MathInline>.</li>
      <li>For small samples, prefer Clopper-Pearson or the Wilson interval.</li>
      <li>The rule of thumb <MathInline>{"np \\geq 5"}</MathInline> and <MathInline>{"n(1-p) \\geq 5"}</MathInline> guides use of the normal approximation.</li>
    </ul>
  </div>
</MdxCard>

import numpy as np
from scipy import stats

p = 0.08
n = 10_000

# i.i.d. default simulation
rng = np.random.default_rng(42)
defaults = rng.binomial(1, p, size=n)

p_hat = defaults.mean()
se = np.sqrt(p_hat * (1 - p_hat) / n)

# Theoretical law (scipy.stats.bernoulli)
bernoulli = stats.bernoulli(p)
print(f"Theoretical E[X]: {bernoulli.mean():.4f}")
print(f"Theoretical Var(X): {bernoulli.var():.4f}")
print(f"Empirical p̂: {p_hat:.4f} (SE ≈ {se:.4f})")

# Approximate Wald 95% CI
z = stats.norm.ppf(0.975)
ci_low = p_hat - z * se
ci_high = p_hat + z * se
print(f"Wald 95% CI: [{ci_low:.4f}, {ci_high:.4f}]")
`}
