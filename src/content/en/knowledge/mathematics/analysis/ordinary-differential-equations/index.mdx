# Ordinary differential equations

<MdxCard title="Definition">
An **ordinary differential equation** (ODE) relates an unknown function <MathInline>{"y(t)"}</MathInline> to its derivatives with respect to a **single** variable (often time):

<MathBlock>{"y' = f(t, y), \\quad y(t_0) = y_0"}</MathBlock>

The **order** is that of the highest derivative. A **Cauchy problem** (initial value problem) fixes <MathInline>{"y(t_0)"}</MathInline> — and possibly derivatives up to order <MathInline>{"n-1"}</MathInline> for an ODE of order <MathInline>{"n"}</MathInline>.
</MdxCard>

## Intuition

<MdxCard>
An ODE describes **dynamic evolution**: at each instant, the rate of change of <MathInline>{"y"}</MathInline> depends on the current state (and possibly time). Knowing <MathInline>{"y_0"}</MathInline> is the starting point; solving the ODE traces the trajectory.

Familiar examples in finance and probability:

- exponential decay (continuous discounting);
- mean reversion (Ornstein-Uhlenbeck);
- logistic growth (saturation).
</MdxCard>

## Solution methods

<MdxCard title="Analytical">
- **Separation of variables** when <MathInline>{"f(t,y) = g(t)h(y)"}</MathInline>.
- **Integrating factor** for first-order linear ODEs.
- **Characteristic equation** for constant-coefficient linear ODEs of order <MathInline>{"n"}</MathInline>.
- **Taylor / power series** for formal solutions — see [Taylor series](/knowledge/mathematics/analysis/taylor-series).
</MdxCard>

<MdxCard title="Numerical">
When no closed formula exists:

<MathBlock>{"y_{n+1} = y_n + h \\, f(t_n, y_n) \\quad \\text{(explicit Euler)}"}</MathBlock>

Runge-Kutta (RK4) and implicit schemes improve stability and accuracy. See [Numerical methods](/knowledge/mathematics/numerical-methods) for finite differences and scheme stability.
</MdxCard>

## Key theorems

<MdxCard>
**Existence and uniqueness (Cauchy-Lipschitz)**: if <MathInline>{"f"}</MathInline> is continuous and Lipschitz in <MathInline>{"y"}</MathInline>, the Cauchy problem has a **unique** local solution.

Consequences:

- no ambiguity in the trajectory for a given initial condition;
- sensitivity to parameters is controlled (variational equations);
- numerical simulators behave predictably if <MathInline>{"h"}</MathInline> is small enough.
</MdxCard>

## Applications

<MdxCard>
- **Finance**: rate models (Vasicek, CIR mean dynamics), continuous discounting, noise-free fund dynamics.
- **Probability**: deterministic ODEs for process moments; bridge to **SDEs** when a diffusion term is added.
- **Physics / engineering**: circuits, mechanics, optimal control (Pontryagin's maximum principle).
- **Machine learning**: Neural ODEs (networks defined as ODEs) — same formalism, numerical backprop through the solver.

[PDEs](/knowledge/mathematics/analysis/partial-differential-equations) generalize the formalism to several independent variables (time + space, or time + market factors).
</MdxCard>
