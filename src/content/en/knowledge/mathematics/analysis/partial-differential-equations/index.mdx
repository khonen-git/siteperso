# Partial differential equations

<MdxCard title="Definition">
A **partial differential equation** (PDE) relates an unknown function <MathInline>{"u(x_1, \\ldots, x_n)"}</MathInline> to its **partial derivatives**:

<MathBlock>{"F\\left(x, u, \\frac{\\partial u}{\\partial x_i}, \\frac{\\partial^2 u}{\\partial x_i \\partial x_j}, \\ldots\\right) = 0"}</MathBlock>

Unlike [ODEs](/knowledge/mathematics/analysis/ordinary-differential-equations), several **independent** variables coexist — typically **time** <MathInline>{"t"}</MathInline> and **space** or **market** variables (underlying price, rates, etc.).

A solution is generally fixed by **boundary conditions** (on the domain boundary) and/or **initial conditions** (at <MathInline>{"t = 0"}</MathInline>).
</MdxCard>

## Intuition

<MdxCard>
A PDE describes a phenomenon varying **simultaneously** in time and space (or across several factors). The solution <MathInline>{"u"}</MathInline> is no longer a curve but a **surface** or field.

- **Elliptic**: equilibrium state, no preferred propagation direction (Laplace).
- **Parabolic**: diffusion / dissipation over time (heat, Black-Scholes).
- **Hyperbolic**: wave propagation at finite speed (wave equation).
</MdxCard>

## Solution methods

<MdxCard title="Analytical">
- **Separation of variables** (Fourier, Sturm-Liouville).
- **Fourier / Laplace transform** for linear equations on simple domains.
- **Green's functions**: solution for a point source.
- **Black-Scholes**: transform to heat equation via change of variables.
</MdxCard>

<MdxCard title="Numerical">
- **Finite differences**: discretize <MathInline>{"u_t"}</MathInline>, <MathInline>{"u_{xx}"}</MathInline> on a grid — see [Numerical methods](/knowledge/mathematics/numerical-methods).
- **Finite elements**: variational basis on complex domains.
- **Monte Carlo**: probabilistic alternative via Feynman-Kac (risk-neutral expectation).

Choice depends on domain, dimension, and boundary conditions.
</MdxCard>

## Probability link: Feynman-Kac

<MdxCard>
For certain parabolic PDEs, the solution is an **expectation**:

<MathBlock>{"u(x, t) = \\mathbb{E}\\left[ e^{-\\int_t^T r(s)\\,ds} \\, g(X_T) \\, \\middle| \\, X_t = x \\right]"}</MathBlock>

where <MathInline>{"X"}</MathInline> follows an SDE tied to the PDE coefficients. This bridges:

- **PDE pricing** (Black-Scholes backward);
- **Monte Carlo pricing** (risk-neutral simulation).

Both approaches should yield the same price under the same assumptions — useful for cross-validation.
</MdxCard>

## Applications

<MdxCard>
- **Quantitative finance**: option pricing ([Black-Scholes](/knowledge/quantitative-finance/options-derivatives/black-scholes)), local vol models, surface calibration.
- **Physics**: diffusion, fluid mechanics, electromagnetism.
- **Engineering**: heat transfer, shape optimization.
- **Image / ML**: diffusion as noise models (score-based models) — same parabolic formalism.

[ODEs](/knowledge/mathematics/analysis/ordinary-differential-equations) are the single independent variable case; PDEs add spatial or multi-factor dimension.
</MdxCard>
