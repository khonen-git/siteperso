Configuer Jupyter Notebook dans Visual Studio Code

Prérequis

Avoir installé les extensions Python et Jupyter dans VSCode.

Créer l'environnement virtuel python (.venv)

Voici le lien de la documentation de VSCode : https://code.visualstudio.com/docs/python/environments

Installer les packages nécessaires

On active l'environnement virtuel

  • Sur Windows : .venv\Scripts\activate
  • Sur MacOS/Linux : source .venv/bin/activate

Ensuite, on installe les packages

  • Jupyter Notebook : pip install jupyter
  • Ipykernel : pip install ipykernel
  • Ajouter les packages nécessaires au projet.
    Exemple : pip install pandas numpy matplotlib

Configuration

Ajouter l'environnement virtuel comme noyau dans Jupyter :
python -m ipykernel install --user --name=.venv --display-name "Python (.venv)"

Voir la liste :
jupyter kernelspec list
