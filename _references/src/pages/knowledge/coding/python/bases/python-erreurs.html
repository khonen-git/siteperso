Gestion des erreurs en Python

Exceptions

Les erreurs en Python sont appelées des exceptions et sont levées lorsqu'une instruction ne peut pas être exécutée correctement :

try:
    # instruction susceptible de lever une exception
except:
    # traitement de l'exception

Blocs try/except

Les blocs 'try'/'except' permettent d'essayer une instruction et de gérer l'exception qui pourrait être levée si l'instruction échoue :

try:
    # instruction susceptible de lever une exception
except NomException:
    # traitement de l'exception

Levée d'exceptions

Les exceptions peuvent être levées manuellement en utilisant l'instruction 'raise' :

if x < 0:
    raise ValueError("x ne peut pas être négatif")

Exceptions personnalisées

Les exceptions peuvent être héritées et personnalisées pour des cas spécifiques :

class MonException(Exception):
    def __init__(self, message):
        super().__init__(message)
        self.message = message

try:
    # instruction susceptible de lever MonException
except MonException as e:
    # traitement de l'exception personnalisée
