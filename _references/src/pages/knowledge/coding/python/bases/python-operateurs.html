Les opérateurs

Opérateurs arithmétiques

Les opérateurs arithmétiques sont utilisés pour effectuer des opérations mathématiques sur les variables en Python. Les opérateurs arithmétiques de base sont:

  • + : Addition - Utilisé pour additionner deux valeurs numériques.
  • - : Soustraction - Utilisé pour soustraire une valeur numérique d'une autre.
  • * : Multiplication - Utilisé pour multiplier deux valeurs numériques.
  • / : Division - Utilisé pour diviser une valeur numérique par une autre.
  • // : Division entière - Utilisé pour effectuer une division entière et renvoyer le quotient entier.
  • % : Modulo - Utilisé pour obtenir le reste d'une division entière.
  • ** : Exponentiation - Utilisé pour calculer une valeur élevée à une puissance donnée.

Opérateurs de comparaison

Les opérateurs arithmétiques sont utilisés pour effectuer des opérations mathématiques sur les variables en Python. Les opérateurs arithmétiques de base sont:

  • < : Inférieur à - Utilisé pour vérifier si une valeur est inférieure à une autre.
  • > : Supérieur à - Utilisé pour vérifier si une valeur est supérieure à une autre.
  • <= : Inférieur ou égal à - Utilisé pour vérifier si une valeur est inférieure ou égale à une autre.
  • >= : Supérieur ou égal à - Utilisé pour vérifier si une valeur est supérieure ou égale à une autre.
  • == : Égal à - Utilisé pour vérifier si deux valeurs sont égales.
  • != : Différent de - Utilisé pour vérifier si deux valeurs ne sont pas égales.

Opérateurs logiques

Les opérateurs arithmétiques sont utilisés pour effectuer des opérations mathématiques sur les variables en Python. Les opérateurs arithmétiques de base sont:

  • and : ET logique - Utilisé pour combiner deux expressions logiques et renvoyer True si les deux sont vraies.
  • or : OU logique - Utilisé pour combiner deux expressions logiques et renvoyer True si au moins l'une d'elles est vraie.
  • not : NON logique - Utilisé pour inverser la valeur booléenne d'une expression.

Opérateurs bit à bit

Les opérateurs arithmétiques sont utilisés pour effectuer des opérations mathématiques sur les variables en Python. Les opérateurs arithmétiques de base sont:

  • << : Décalage à gauche - Utilisé pour décaler les bits d'un nombre vers la gauche.
  • >> : Décalage à droite - Utilisé pour décaler les bits d'un nombre vers la droite.
  • & : ET bit à bit - Utilisé pour effectuer une opération "ET" bit à bit entre deux nombres.
  • | : OU bit à bit - Utilisé pour effectuer une opération "OU" bit à bit entre deux nombres.
  • ^ : OU exclusif bit à bit - Utilisé pour effectuer une opération "OU exclusif" bit à bit entre deux nombres.
  • ~ : Inversion de bits - Utilisé pour inverser tous les bits d'un nombre.

Opérateur d'affectation

Les opérateurs arithmétiques sont utilisés pour effectuer des opérations mathématiques sur les variables en Python. Les opérateurs arithmétiques de base sont:

  • := : Affectation de variable (aussi appelé "walrus operator") - Utilisé pour affecter une valeur à une variable tout en évaluant une expression conditionnelle. Il est principalement utilisé dans les boucles et les structures de contrôle.

Opérateur de décorateur

Les opérateurs arithmétiques sont utilisés pour effectuer des opérations mathématiques sur les variables en Python. Les opérateurs arithmétiques de base sont:

  • @ : Décorateur - Utilisé dans Python pour appliquer des décorateurs aux fonctions et aux méthodes. Il est utilisé dans le contexte de la programmation orientée objet pour modifier le comportement des fonctions ou des méthodes.

Exemples d'utilisation des opérateurs arithmétiques

Voici quelques exemples d'utilisation des opérateurs arithmétiques en Python:


x = 10
y = 5

# addition
print(x + y)  # affiche 15

# soustraction
print(x - y)  # affiche 5

# multiplication
print(x * y)  # affiche 50

# division
print(x / y)  # affiche 2.0

# modulo
print(x % y)  # affiche 0

# division entière
print(x // y)  # affiche 2

# exponentiation
print(x ** 2)  # affiche 100
