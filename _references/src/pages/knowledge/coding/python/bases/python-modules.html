Les modules en Python

Introduction

Un module est un fichier contenant du code Python, qui peut être importé et utilisé dans d'autres programmes :

# importer le module 'math'
import math

# utiliser la fonction 'sqrt' du module 'math'
x = math.sqrt(4)

Utilisation des modules

Les fonctions et les classes définies dans un module peuvent être utilisées en appelant leur nom précédé du nom du module :

# importer le module 'math'
import math

# utiliser la fonction 'sqrt' du module 'math'
x = math.sqrt(4)

# utiliser la constante 'pi' du module 'math'
y = math.pi

Les modules standards

Les modules standards sont inclus dans l'installation de Python :

# importer le module 'os'
import os

# utiliser la fonction 'getcwd' du module 'os'
x = os.getcwd()

Les modules tiers

Les modules tiers doivent être installés séparément :

# installer le module 'pandas'
!pip install pandas

# importer le module 'pandas'
import pandas as pd

# utiliser la fonction 'read_csv' du module 'pandas'
df = pd.read_csv('data.csv')
