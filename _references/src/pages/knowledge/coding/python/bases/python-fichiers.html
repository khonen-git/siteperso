Fichiers en Python

Ouvrir et fermer un fichier

Pour travailler avec des fichiers, il faut d'abord ouvrir le fichier en utilisant la fonction 'open()'. Les fichiers ouverts doivent être fermés avec la méthode 'close()' pour libérer les ressources du système :

fichier = open("mon_fichier.txt", "r")  # ouvrir en mode lecture
# ... travailler avec le fichier ...
fichier.close()  # fermer le fichier

Lire un fichier ligne par ligne

Les fichiers peuvent être lus ligne par ligne avec la méthode 'readline()' :

fichier = open("mon_fichier.txt", "r")  # ouvrir en mode lecture
for ligne in fichier:
    print(ligne)
fichier.close()  # fermer le fichier

Lire ou écrire un fichier en blocs

Les fichiers peuvent être lus ou écrits en blocs avec les méthodes 'read()' et 'write()' :

fichier = open("mon_fichier.txt", "r")  # ouvrir en mode lecture
contenu = fichier.read()  # lire tout le contenu du fichier
fichier.close()  # fermer le fichier

fichier = open("mon_fichier.txt", "w")  # ouvrir en mode écriture
fichier.write("Bonjour, monde!")  # écrire du contenu dans le fichier
fichier.close()  # fermer le fichier

Mode d'ouverture des fichiers

Les fichiers peuvent être ouverts en mode lecture ('r'), écriture ('w') ou ajout ('a') :

fichier = open("mon_fichier.txt", "r")  # ouvrir en mode lecture
fichier = open("mon_fichier.txt", "w")  # ouvrir en mode écriture
fichier = open("mon_fichier.txt", "a")  # ouvrir en mode ajout
