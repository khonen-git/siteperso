Chaînes de caractères en Python

Définition des chaînes de caractères

Les chaînes de caractères sont des séquences de caractères, définies entre guillemets simples ('...') ou doubles ("...") :


ma_chaine = "Hello, world!"
print(ma_chaine)

Concaténation de chaînes de caractères

Les chaînes de caractères peuvent être concaténées avec l'opérateur '+' :


chaine1 = "Bonjour"
chaine2 = "le monde"
ma_chaine = chaine1 + ", " + chaine2 + "!"
print(ma_chaine)

Formatage des chaînes de caractères

Les chaînes de caractères peuvent être formatées avec la méthode 'format()' :

nom = "Dupont"
prenom = "Jean"
age = 30
ma_chaine = "Je m'appelle {} {}, j'ai {} ans.".format(prenom, nom, age)
print(ma_chaine)

Méthodes des chaînes de caractères

Les chaînes de caractères ont de nombreuses méthodes intégrées, telles que 'upper()', 'lower()', 'strip()', 'split()', etc. :

ma_chaine = "   Hello, world!   "
print(ma_chaine.upper())  # mettre en majuscules
print(ma_chaine.lower())  # mettre en minuscules
print(ma_chaine.strip())  # supprimer les espaces en début et fin de chaîne
print(ma_chaine.split(", "))  # diviser la chaîne en une liste de sous-chaînes
