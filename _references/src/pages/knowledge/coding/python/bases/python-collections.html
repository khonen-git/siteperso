Collections de données en Python

Listes

Les listes sont des collections ordonnées et modifiables d'éléments :

ma_liste = [1, 2, 3, "hello", "world"]
print(ma_liste)

# modifier un élément
ma_liste[0] = "bonjour"
print(ma_liste)

# ajouter un élément à la fin de la liste
ma_liste.append(4)
print(ma_liste)

# supprimer un élément de la liste
ma_liste.remove("hello")
print(ma_liste)

Tuples

Les tuples sont des collections ordonnées et immuables d'éléments :

mon_tuple = (1, 2, 3, "hello", "world")
print(mon_tuple)

# accéder à un élément
print(mon_tuple[0])

# lever une exception en cas de tentative de modification
mon_tuple[0] = "bonjour"  # lève une exception

Ensembles

Les ensembles sont des collections non ordonnées et non indexées d'éléments uniques :

mon_ensemble = {1, 2, 3, 4, 5}
print(mon_ensemble)

# ajouter un élément
mon_ensemble.add(6)
print(mon_ensemble)

# supprimer un élément
mon_ensemble.remove(2)
print(mon_ensemble)

Dictionnaires

Les dictionnaires sont des collections non ordonnées et modifiables d'éléments, composées de paires clé-valeur :

mon_dictionnaire = {"prenom": "Jean", "nom": "Dupont", "age": 30}
print(mon_dictionnaire)

# accéder à une valeur par clé
print(mon_dictionnaire["prenom"])

# modifier une valeur
mon_dictionnaire["age"] = 31
print(mon_dictionnaire)

# ajouter une nouvelle paire clé-valeur
mon_dictionnaire["ville"] = "Paris"
print(mon_dictionnaire)
