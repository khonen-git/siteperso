Variables en Python

Types de données en Python

En Python, les types de données les plus courants sont:

  • Nombres entiers (int)
  • Nombres à virgule flottante (float)
  • Chaînes de caractères (str)
  • Listes (list)
  • Tuples (tuple)
  • Dictionnaires (dict)
  • Ensembles (set)
  • Booléens (bool)
  • Objets None (NoneType)

Règles pour nommer les variables

Les variables en Python doivent respecter certaines conventions de nommage. Les noms de variables doivent être en minuscules, avec des mots séparés par des underscores (_). Les noms de variables ne doivent pas commencer par des chiffres, ni contenir d'espaces. Les noms de variables doivent être descriptifs et concis. Par exemple:

ma_variable = 42
nom_complet = "John Smith"
liste_de_nombres = [1, 2, 3, 4]

Portée des variables

En Python, la portée d'une variable peut être locale ou globale. Les variables définies à l'intérieur d'une fonction ont une portée locale, tandis que les variables définies à l'extérieur d'une fonction ont une portée globale. Par exemple:

x = 42  # variable globale

def foo():
  x = 21  # variable locale
  print(x)

foo()  # affiche 21
print(x)  # affiche 42

Opérations de base sur les variables

En Python, vous pouvez effectuer plusieurs opérations sur les variables, notamment:

  • Affectation: x = 42
  • Modification: x = x + 1
  • Suppression: del x

Conversions de type

En Python, vous pouvez convertir une variable d'un type de données à un autre en utilisant des fonctions intégrées telles que int(), float(), str(), etc. Par exemple:

x = "42"
print(int(x))  # convertit x en un entier

Fonctions intégrées liées aux variables

Python offre de nombreuses fonctions intégrées qui peuvent être utilisées avec des variables. Voici quelques exemples:

  • len(): renvoie la longueur d'une chaîne de caractères ou d'une liste
  • str(): convertit un objet en une chaîne de caractères
  • int(): convertit un objet en un nombre entier
  • float(): convertit un objet en un nombre à virgule flottante
  • type(): renvoie le type de données d'un objet

Voici quelques exemples d'utilisation:

ma_liste = [1, 2, 3, 4, 5]
ma_chaine = "Hello, World!"
mon_nombre = 42

print(len(ma_liste))  # affiche 5
print(str(mon_nombre))  # affiche "42"
print(int(3.14))  # affiche 3
print(float(42))  # affiche 42.0
print(type(ma_chaine))  # affiche <class 'str'>
