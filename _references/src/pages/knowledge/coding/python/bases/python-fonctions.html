Les fonctions en Python

Définition d'une fonction

Pour définir une fonction en Python, on utilise le mot-clé def, suivi du nom de la fonction et des paramètres entre parenthèses :

def nom_de_la_fonction(parametre1, parametre2, ...):
    # bloc de code à exécuter
    return resultat

Arguments de fonction

Les fonctions peuvent prendre des arguments positionnels et/ou nommés. Les paramètres peuvent également avoir une valeur par défaut :

def ma_fonction(arg1, arg2=0, arg3=""):
    # bloc de code à exécuter
    return resultat

# appel de la fonction avec des arguments positionnels
ma_fonction(10, 20)

# appel de la fonction avec des arguments nommés
ma_fonction(arg1=10, arg2=20)

# appel de la fonction avec un argument nommé et un argument positionnel
ma_fonction(10, arg3="hello")

Fonctions récursives

Une fonction récursive est une fonction qui s'appelle elle-même. Elle est utile pour résoudre des problèmes qui peuvent être divisés en sous-problèmes plus petits :

def factorielle(n):
    if n == 0:
        return 1
    else:
        return n * factorielle(n-1)

Fonctions anonymes (lambda)

Les fonctions anonymes (ou lambda) sont des fonctions sans nom qui peuvent être utilisées pour des opérations simples :

ma_fonction = lambda x: x * 2

Fonctions intégrées

Python dispose d'un grand nombre de fonctions intégrées, telles que print(), len(), range(), sum(), etc. Voici quelques exemples d'utilisation :

a = [1, 2, 3, 4, 5]
# utiliser la fonction print() pour afficher du texte à l'écran
print("Hello, world!")

# utiliser la fonction len() pour obtenir la longueur d'une liste
a = [1, 2, 3, 4, 5]
print(len(a))

# utiliser la fonction range() pour créer une liste d'entiers
b = range(10)
print(list(b))

# utiliser la fonction sum() pour calculer la somme d'une liste de nombres
c = [1, 2, 3, 4, 5]
print(sum(c))
