Opérations basiques

Les structures de données de base fournies par Polars sont les séries (Series) et les tableaux de données (DataFrames).

Series

Les Series sont une structure de données unidimensionnelle. Au sein d'une série, tous les éléments ont le même type de données. 


import polars as pl

# Créer une Series de nombres
numbers_series = pl.Series("numbers", [1, 2, 3, 4, 5])

# Afficher la Series numbers_series
print(numbers_series)

# Créer une Series de chaînes de caractères
letters_series = pl.Series("letters", ["a", "b", "c", "d", "e"])

# Afficher la Series letters_series
print(letters_series)

DataFrames

La DataFrame est une structure de données bidimensionnelle qui est soutenue par une série (Series), et peut être vue comme une abstraction d'une collection (par exemple, une liste) de séries. Les opérations qui peuvent être exécutées sur un DataFrame sont très similaires à ce qui est fait dans une requête de type SQL. On peut effectuer des GROUP BY, JOIN, PIVOT, mais aussi définir des fonctions personnalisées. 


import polars as pl

# Exemple de DataFrame
df = pl.DataFrame({
    "A": [1, 2, 3, 4],
    "B": ["a", "b", "c", "d"]
})

# Afficher la DataFrame
print(df)
